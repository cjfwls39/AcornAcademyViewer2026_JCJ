transform: translate 트랜스폼

특정 요소의 크기나 형태가 변하는 것을 변형(변환) 또는 트랜스폼이라고 한다.
사용자의 동작에 반응해서 텍스트나 이미지 등을 움직이게 할 수 있다.
2차원(x, y) / 3차원(x, y, z) : 원근감의 표현
z축 앞뒤로 이동하는데 가까워지고 멀어진다.
위치이동 : translate
확대/축소 : scale
회전 : rotate
기울기(비틀기) : skew

transition 전환 : 속성의 변화
transform 변환 : translate, skew, rotate3e, scale
애니메이션효과 1회만 적용
